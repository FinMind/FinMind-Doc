#### 外幣對台幣資料表 TaiwanExchangeRate

- 針對 USD 做示範

!!! example
     ```python
     import requests
     import pandas as pd
     url = "https://api.finmindtrade.com/api/v4/data"
     parameter = {
          "dataset": "TaiwanExchangeRate",
          "data_id": "USD",
          "start_date": "2006-01-01",
     }
     data = requests.get(url, params=parameter)
     data = data.json()
     data = pd.DataFrame(data['data'])
     print(data.head())
     ```
