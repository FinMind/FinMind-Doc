# 台股指數代號對照表

`taiwan_stock_tick_snapshot` 除了支援一般股票代號（4 碼）之外，也支援台股大盤、產業類股、主題指數等共 91 個指數代號（3 碼）。

例如 `data_id="001"` 即為加權指數，`data_id="101"` 為櫃買加權。

下表為完整對照清單（依用途分組），名稱沿用台灣證券交易所（TWSE）/ 證券櫃檯買賣中心（TPEx）原始公告之指數中文名稱。

## 使用範例

!!! example
    === "Package"
        ```python
        from FinMind.data import DataLoader

        api = DataLoader()
        # api.login_by_token(api_token='token')
        df = api.taiwan_stock_tick_snapshot(stock_id="001")  # 加權指數
        # 一次拿多個（個股 + 指數可混搭）
        df = api.taiwan_stock_tick_snapshot(stock_id=["001", "101", "2330"])
        # 一次拿全部
        df = api.taiwan_stock_tick_snapshot()
        ```
    === "Python-request"
        ```python
        import requests
        import pandas as pd
        token = ""  # 參考登入，獲取金鑰
        headers = {"Authorization": f"Bearer {token}"}
        url = "https://api.finmindtrade.com/api/v4/taiwan_stock_tick_snapshot"
        parameter = {
            "data_id": "001",  # 加權指數
            # "data_id": ["001", "101"],  # 加權 + 櫃買，一次拿多個
        }
        resp = requests.get(url, headers=headers, params=parameter)
        data = pd.DataFrame(resp.json()["data"])
        print(data)
        ```
