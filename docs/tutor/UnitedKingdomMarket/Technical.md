#### 英國股票清單列表 UKStockInfo

- 這張資料表主要是列出英國所有上市上櫃的股票名稱，代碼和產業類別

!!! example
   ```python
   import requests
   import pandas as pd

   url = 'https://api.finmindtrade.com/api/v4/data'
   parameter = {
      "dataset": "UKStockInfo"
   }
   data = requests.get(url, params=parameter)
   data = data.json()
   data = pd.DataFrame(data['data'])
   print(data.head())
   ```
