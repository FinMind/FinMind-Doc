# 隱私權政策

本政策說明 FinMind 在透過 API 及 AI 工具（例如 Custom GPT、MCP server 等）提供金融資料查詢服務時，如何處理相關資料。使用本服務即表示您瞭解並同意本政策。

## 適用範圍

本政策適用於您使用 FinMind 對外提供之資料查詢介面。本服務所提供之內容僅供教育與參考用途，資料正確性與使用結果請參閱[使用條款](License.md)。

## 處理的資料

為回應您的查詢，本服務會處理您所送出的查詢內容（例如資料集名稱、標的代號與日期區間），以及用於驗證存取權限的存取憑證（token）。

## 資料用途

上述資料僅用於下列目的：

- 回應您的查詢並提供對應的金融資料
- 驗證存取權限與套用對應的使用方案
- 服務之維運、安全維護與品質改善

## 我們不會做的事

- 不販售您的個人資料
- 不將資料用於與本服務無關之目的

## 第三方平台

當您透過第三方 AI 平台（例如 ChatGPT）使用本服務時，您與該平台之間的互動另受該平台自身之隱私權政策規範，不在本政策範圍之內。

## 資料安全

本服務採取合理之技術與管理措施，以保護所處理之資料。請妥善保管您的存取憑證，避免外洩。

## 政策更新

本政策得因服務調整而不時更新，更新後之版本將公告於本頁。
